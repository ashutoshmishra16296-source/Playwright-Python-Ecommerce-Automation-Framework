# Playwright Python Automation Framework

## Overview

This is a Playwright + Pytest automation framework built using the Page Object Model (POM).

## Features

- Playwright
- Pytest
- Page Object Model
- Base Page
- CSV Data Driven Testing
- HTML Reports
- Screenshot on Failure
- Logging
- Config Reader
- Retry Failed Tests

## Project Structure

```
AutomationProjectsEcommerceAutomationFramework

config/

screenshots/
logs/

pytest.ini
README.md
```

## Installation

```bash
pip install -r requirements.txt
```

## Run Tests

```bash
pytest
```
